Esse projeto está sendo desenvolvido como treinamento da sintaxe do PHP.

O objetivo é criar um sistema que gerencia o hardware de uma empresa, 
garantindo que você saiba onde cada equipamento está e qual seu estado de conservação.

O projeto simula o controle de ativos de uma empresa (notebooks, servidores), lidando com estados e herança.

O sistema deve ter persistência de dados, utilizando JSON como "banco de dados".

Requisitos Funcionais:

- O sistema deve ser capaz de listar todos os equipamentos cadastrados
- O usuário técnico deve ser capaz de alterar o status de um equipamento
- O usuário técnico deve ser capaz de cadastrar novos equipamentos

Requisitos Não Funcionais:

- Os dados devem ser armazenados em um arquivo JSON
- O sistema deve permitir o cadastro de equipamentos do tipo Notebook e Servidor
